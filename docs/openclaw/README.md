## Über OpenClaw

**OpenClaw** ist ein lokal laufender, selbständiger KI-Agent – dein persönlicher Assistent, der tatsächlich Dinge erledigt:

- ✅ Clears your inbox
- ✅ Sends emails
- ✅ Manages your calendar
- ✅ Checks you in for flights
- ✅ **All from WhatsApp, Telegram, or any chat app**

**Key Differentiator:** Private by default – deine Daten bleiben auf deinem Gerät.

→ [Offizielle Website](https://openclaw.ai/) | [GitHub](https://github.com/openclaw/openclaw) | [Discord](https://discord.com/invite/clawd)
