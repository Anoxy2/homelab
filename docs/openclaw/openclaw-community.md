## Beitragen

### Als Nutzer

1. **Skills teilen** → Auf ClawHub veröffentlichen
2. **Feedback geben** → Discord oder GitHub Issues
3. **Dokumentation verbessern** → PRs willkommen
4. **Anderen helfen** → Discord #help

### Als Entwickler

```bash
# Fork & Clone
git clone https://github.com/yourname/openclaw.git

# Branch erstellen
git checkout -b feature/amazing-feature

# Änderungen committen
git commit -m "Add amazing feature"

# PR erstellen
git push origin feature/amazing-feature
```

### Skill Contribution

```bash
# Skill entwickeln
mkdir my-skill
cd my-skill
# ... Implementierung ...

# Auf ClawHub veröffentlichen
openclaw plugins publish .
```

## Community-Guidelines

### Lobster Code of Conduct 🦞

1. **Sei respektvoll** – Konstruktive Kritik
2. **Hilf anderen** – Knowledge sharing
3. **Credit geben** – Original-Autoren erwähnen
4. **Sicherheit first** – Keine Secrets teilen
5. **Have fun** – AI-Assistenten sind cool!

### Moderation

| Stufe | Konsequenz |
|-------|------------|
| Verwarnung | DM von Moderator |
| Timeout | Temporärer Ausschluss |
| Kick | Entfernung |
| Ban | Permanenter Ausschluss |
