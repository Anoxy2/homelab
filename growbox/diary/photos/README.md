# Growbox Foto-Ritual

Woechentlich mindestens ein Foto im Format `DD.MM.YYYY.jpg` ablegen.

Beispiel:
- `05.04.2026.jpg`

Hinweise:
- Motiv moeglichst konstant (gleicher Blickwinkel/Licht), damit Veraenderungen vergleichbar sind.
- Optional kann statt `.jpg` auch `.jpeg` oder `.png` genutzt werden.
